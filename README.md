# 📊 Student Performance Analysis

## 📌 Project Overview
This project analyzes student performance using Python. It explores marks in Math, Reading, and Writing and identifies patterns in student performance.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## 📊 Features
- Data cleaning and preprocessing
- Average score calculation
- Data visualization using charts
- Performance comparison

## 📈 Insights
- Reading and writing scores are generally higher than math
- Most students score between 60–80 range
- Strong correlation between subjects

## 🚀 Conclusion
This project demonstrates basic data analysis skills using Python.
